# Mailgun

Mailgun module provides integration with the [Mailgun](https://www.mailgun.com/)
email service. The module uses Mailgun's official [SDK for PHP](https://github.com/mailgun/mailgun-php).

Mailgun is email automation created for developers. It uses REST APIs to
effortlessly send, receive and track emails.

## Mailgun features

  - [Sending Email](https://www.mailgun.com/products/send/)
  - [Logs](https://www.mailgun.com/features/email-logs/)
  - [Analytics](https://www.mailgun.com/features/email-analytics/)
  - [Deliverability](https://www.mailgun.com/solutions/email-deliverability-service/)

Please [get acquainted with Mailgun prices](https://www.mailgun.com/pricing)
before using this module.

## Installation, Configuration and Usage

  - Install this module and its dependencies using the official
    [Backdrop CMS instructions](https://backdropcms.org/guide/modules)
  - The Mailgun SDK library needs to be added as well. At the moment we are
    following the recommendation from the Drupal 7 version to use the
    [Composer Manager](https://backdropcms.org/project/composer_manager) module
    to manage external library dependencies, but you could also load the library
    separately if you are managing Composer dependencies differently, so the
    Composer Manager module is not a hard requirement.
  - View and edit more instructions in the
    [Wiki](https://github.com/backdrop-contrib/mailgun/wiki).
